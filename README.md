# Trip

Trip is a travel planning project with a React + Vite frontend under `Frontend/FrontEnd`.

## Project Structure

- `Frontend/FrontEnd`: main frontend application
- `package.json`: root-level shared dependencies

## Getting Started

### Root dependencies

```powershell
npm install
```

### Frontend app

```powershell
cd Frontend/FrontEnd
npm install
npm run dev
```

## Frontend Scripts

Run these inside `Frontend/FrontEnd`:

```powershell
npm run dev
npm run build
npm run lint
npm run preview
```

## Notes

- Environment files are ignored by Git via `.gitignore`
- `node_modules` and build outputs are excluded from version control
